Introduction to I Wanna Be The Guy Fangames

Setup

  • Be sure to extract each game's zip into its own folder.
  • The default in-game controls are Shift to jump/select, Z to shoot, R to respawn, and Escape to quit. Some games have in-game control rebinding, but for the rest you'll have to use a program like JoyToKey if you want to rebind your controls.
